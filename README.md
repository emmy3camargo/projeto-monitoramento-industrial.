# 🪵 Monitoramento de Secagem Industrial - Simulação Python

Este projeto une minha experiência prática como **Auxiliar de Produção (Setor Secador)** na Indústria Sudati com os conhecimentos que estou adquirindo em **Engenharia de Software**.

## 🎯 Objetivo
O sistema simula o monitoramento térmico de um secador industrial de lâminas de madeira. O foco é demonstrar como a tecnologia pode auxiliar na segurança operacional e no controle de qualidade, automatizando a identificação de temperaturas fora do padrão.

## 🚀 Funcionalidades
- **Leitura em Tempo Real:** Simulação de sensores de temperatura.
- **Alertas de Segurança:** Notificações automáticas para temperaturas críticas (risco de incêndio ou dano ao material).
- **Lógica de Processo:** Baseado nos padrões reais de secagem industrial.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python 3
- **Conceitos:** Lógica de programação, estruturas condicionais e automação.
